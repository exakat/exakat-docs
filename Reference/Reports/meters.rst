The Meters report export various dimensions of the audited code.

Exakat measures a large number of code dimensions, such as number of files, lines of code, tokens. All those are collected in this report.

::

    {
    	"loc": 95950,
    	"locTotal": 140260,
    	"files": 1824,
    	"tokens": 677213
    }
