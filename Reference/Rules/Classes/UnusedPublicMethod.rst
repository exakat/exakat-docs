Unused Public Methods
+++++++++++++++++++++

  List unused public methods.

.. code-block:: php
   
   <?php
   
   class x {
   	public function usedMethod() {}
   	
   	// There is no call to this method
   	public function unusedMethod() {}
   }
   
   $x = new x();
   $x->usedMethod();
   
   
   ?>
