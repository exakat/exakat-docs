Dynamically Called Classes
++++++++++++++++++++++++++

  Indicates if a class is called dynamically.

.. code-block:: php
   
   <?php
   
   // This class is called dynamically
   class X {
       const CONSTANTE = 1;
   }
   
   $classe = 'X';
   
   $x = new $classe();
   
   echo $x::CONSTANTE;
   
   ?>
