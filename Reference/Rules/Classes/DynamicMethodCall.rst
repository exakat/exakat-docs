Dynamic Methodcall
++++++++++++++++++

  Dynamic calls to class methods.

.. code-block:: php
   
   <?php
   
   class x {
       static public function foo() {}
              public function bar() {}
   }
   
   $staticmethod = 'foo';
   // dynamic static method call to x::foo()
   x::$staticmethod();
   
   $method = 'bar';
   // dynamic method call to bar()
   $object = new x();
   $object->$method();
   
   ?>
