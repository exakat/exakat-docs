Useless Constructor
+++++++++++++++++++

  Class constructor that have empty bodies are useless. They may be removed, if they are not called. 


.. code-block:: php
   
   <?php
   
   class X {
       function __construct() {
           // Do nothing
       }
   }
   
   class Y extends X {
       // Useful constructor, as it prevents usage of the parent
       function __construct() {
           // Do nothing
       }
   }
   
   ?>

Suggestions
___________

* Remove the constructor
