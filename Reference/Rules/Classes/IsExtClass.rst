Is An Extension Class
+++++++++++++++++++++

  Those classes belongs to a PHP Extensions.

.. code-block:: php
   
   <?php
   
   // This is a native PHP class
   $o = new Stdclass();
   
   // This is not a native PHP class
   $o = new Elephpant();
   
   ?>
