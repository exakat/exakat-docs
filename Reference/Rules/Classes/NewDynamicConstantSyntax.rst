New Dynamic Class Constant Syntax
+++++++++++++++++++++++++++++++++

  There is a dedicated syntax to access dynamically to class constant values. 


.. code-block:: php
   
   <?php
   
   class x {
   	const A = 1;
   }
   
   $a = 'A';
   echo x::{$a}; // displays 1
   
   ?>
