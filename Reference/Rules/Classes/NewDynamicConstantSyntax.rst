PHP has a dedicated syntax to access dynamically class constant values. This was added in PHP 8.3. It prevents using the a call to the function ``constant()``.

.. code-block:: php
   
   <?php
   
   class x {
   	const A = 1;
   }
   
   $a = 'A';
   echo x::{$a}; // displays 1
   
   ?>
