Static Methods
++++++++++++++

  List of all `static <https://www.php.net/manual/en/language.oop5.static.php>`_ methods.


.. code-block:: php
   
   <?php
   
   class foo {
       static public function staticMethod() {
           
       }
       
       public function notStaticMethod() {
       
       }
              
       private function method() {
           // This is not a property
           new static();
       }
   }
   
   ?>
