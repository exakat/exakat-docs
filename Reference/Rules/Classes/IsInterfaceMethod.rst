Is Interface Method
+++++++++++++++++++

  Mark a method as part of an interface that the current class implements.

.. code-block:: php
   
   <?php
   
   interface i {
       function i20();
   }
   
   class x implements i {
       // This is an interface method
       function i20() {}
   
       // This is not an interface method
       function x20() {}
   }
   
   ?>
