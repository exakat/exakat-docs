Cant Instantiate Non Class
++++++++++++++++++++++++++

  It is not possible to instantiate anything else than a class. Interfaces, enumerations and traits cannot be instantiated.

.. code-block:: php
   
   <?php
   
   class c {} 
   
   $object = new c;
   
   trait t {}
   new t;
   
   ?>
