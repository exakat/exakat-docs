PHP 7.2 introduced the ability to remove a type when overloading a method. This is not valid code for older versions.

.. code-block:: php
   
   <?php
   
   class foo {
       function foobar(foo $a) {}
   }
   
   class bar extends foo {
       function foobar($a) {}
   }
   
   ?>
