Defined Properties
++++++++++++++++++

  List of properties that are explicitly defined in the class, its parents or traits.

.. code-block:: php
   
   <?php
   
   class foo {
       // property definition
       private bar = 2;
   }
   
   ?>

See also `Properties <https://www.php.net/manual/en/language.oop5.properties.php>`_.
