Abstract Methods Usage
++++++++++++++++++++++

  List of all abstract methods being used.


.. code-block:: php
   
   <?php
   
   // abstract class
   abstract class foo {
       // abstract method
       function foobar(); 
   }
   
   class bar extends foo {
       // extended abstract method
       function foobar() {
           // doSomething()
       }
   
       // extra method
       function barbar() {
           // doSomething()
       }
   }
   ?>

See also `Classes abstraction <https://www.php.net/abstract>`_.
