Functions Glossary
++++++++++++++++++

  List of all the defined functions in the code.


.. code-block:: php
   
   <?php
   
   // A function
   function aFunction() {}
   
   // Closures (not reported)
   $closure = function ($arg) {  }
   
   // Methods
   class foo {
       function aMethod() {}
   }
   
   ?>
