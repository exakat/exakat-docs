Is Generator
++++++++++++

  Mark as such functions or methods that are using yield and yield from.


.. code-block:: php
   
   <?php
   
   function generator() {
       yield from generator2();
       
       return 3;
   }
   
   function generator2() {
       yield 1;
       yield 2;
   }
   
   ?>

See also `Generators overview <https://www.php.net/manual/en/language.generators.overview.php>`_.
