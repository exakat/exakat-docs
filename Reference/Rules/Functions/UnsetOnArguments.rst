Unset Arguments
+++++++++++++++

  There is no need to unset arguments. Those values will be freed at the end of the function anyhow.

.. code-block:: php
   
   <?php
   
   function foo($a, $b) {
       $b = $a * 2;
       // This is useless. $a will be freed at the end of the function.
       unset($a);
   }
   
   ?>
