Dynamic Function Call
+++++++++++++++++++++

  Mark a functioncall made with a variable name.


.. code-block:: php
   
   <?php
   
   // function definition
   function foo() {}
   
   // function name is in a variable, as a string.
   $var = 'foo'; 
   
   // dynamic call of a function
   $var();
   
   call_user_func($var);
   
   ?>
