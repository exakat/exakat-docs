Identity
++++++++

  This method, function or `closure <https://www.php.net/`closure <https://www.php.net/closure>`_>`_ returns one of its argument, without modification. 


.. code-block:: php
   
   <?php
   
   function foo($a) {
       return $a;
   }
   
   ?>
