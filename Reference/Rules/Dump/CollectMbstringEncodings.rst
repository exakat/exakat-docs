Collect Mbstring Encodings
++++++++++++++++++++++++++

  This analysis collects the encoding names, used by ext/mb functions. 


.. code-block:: php
   
   <?php
   
   mb_stotolower('PHP', 'iso-8859-1');
   
   mb_stotolower('PHP', 'iso-8859-1');
   
   ?>
