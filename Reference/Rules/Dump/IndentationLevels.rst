Indentation Levels
++++++++++++++++++

  Collect all level of indentations for methods and functions.


.. code-block:: php
   
   <?php
   
   function foo() {
       $a = 1; // level 1
       if ($b == 2) {
           $c = 1; // level 2
       }
       $d = 4; // level 1
   }
   
   ?>
