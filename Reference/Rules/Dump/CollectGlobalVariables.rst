Collect Global Variables
++++++++++++++++++++++++

  This rule collects the names of the global variables. The global variables are collected from ``$GLOBALS`` usage, ``global`` keyword usage and variables in the global space.

.. code-block:: php
   
   <?php
   
   global $x;
   
   $GLOBALS['y'] = 3;
   
   ?>
