Collect Class Traits Counts
+++++++++++++++++++++++++++

  Counts the number of trait used in a class. 


.. code-block:: php
   
   <?php
   
   // Use no traits
   class x {}
   
   // Use one trait
   class y {
       use TraitT;
   }
   
   ?>
