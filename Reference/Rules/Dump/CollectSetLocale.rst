Collect SetLocale
+++++++++++++++++

  This rule collects the second argument to all the calls to `setlocale() <https://www.php.net/setlocale>`_. This gives an overview of the which special locales are used in the code.

.. code-block:: php
   
   <?php
   
   setlocale(LC_MONEY, 'nl_nl');
   
   ?>
