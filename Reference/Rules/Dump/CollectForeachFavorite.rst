Foreach() Favorite
++++++++++++++++++

  Collect the name used in `foreach() <https://www.php.net/manual/en/control-structures.foreach.php>`_ loops. Then, sorts them in order of popularity.

.. code-block:: php
   
   <?php
   
   // collect $k, $v
   foreach($array as $k => $v) { }
   
   // collect $k, $v1, $v2
   foreach($array as $k => [$v1, $v2]) { }
   
   ?>
