Collects Variables
++++++++++++++++++

  This rule collects all variables from the code. Their type is mentionned, as variable, object or array, depending on their usage.

.. code-block:: php
   
   <?php
   
   // variable : array
   $array[1] = 2;
   
   // variable : variable
   $value = 3;
   
   // variable : object
   $object->property;
   
   ?>

-------------------
