Collect Literals
++++++++++++++++

  Collects all literals in the application. Strings, integer, float are collected. Booleans, null and arrays are not.

.. code-block:: php
   
   <?php
   
   $a = 1; 
   
   // The array is not collected, but the C and 4 are.
   $b = ['C' => 4];
   ?>
