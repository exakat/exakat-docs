Inclusions
++++++++++

  Collect inclusions of files. This is based on include(), require(), include_once() and require_once() keywords.


.. code-block:: php
   
   <?php
   // This is file 'A.php';
   
   include 'B.php';
   
   // Here, B.php includes A.php
   
   ?>
