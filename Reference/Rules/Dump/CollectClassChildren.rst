Collect Class Children Count
++++++++++++++++++++++++++++

  Count the number of class children for each class.


.. code-block:: php
   
   <?php
   
   // 2 children
   class a {}
   
   // 1 children
   class b extends a {}
   
   // no children
   class c extends b {}
   
   // no children
   class d extends a {}
   ?>
