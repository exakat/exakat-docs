Hexadecimal Glossary
++++++++++++++++++++

  List of all the integer values, written in the hexadecimal format.

.. code-block:: php
   
   <?php
   
   $hexadecimal = 0x10;
   
   $anotherHexadecimal =0XAF;
   
   ?>
