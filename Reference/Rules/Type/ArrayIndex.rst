Type Array Index
++++++++++++++++

  All literal index used in the code.

.. code-block:: php
   
   <?php
   
   // index is an index. it is read
   $array['index'] = 1;
   
   // another_index and second_level are read
   $array[] = $array['another_index']['second_level'];
   
   // variables index are not reported
   $array[$variable] = 1;
   
   ?>
