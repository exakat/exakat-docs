Ip
++

  This rule lists hardocded IPs in the source. Such IPs cannot be changed, and may produce unexpected results. 

.. code-block:: php
   
   <?php
   
   $ip = '123.34.56.227';
   $a = '3627734755';
   $a = '000000000330.0000000072.00000000326.0343';
   
   ?>

See also `IP converter <https://h.43z.one/ipconverter/>`_.
