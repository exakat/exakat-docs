Nested Match
++++++++++++

  Nested match calls makes the code difficult to read. It is recommended to avoid nesting match calls.

.. code-block:: php
   
   <?php
   
   $a = match($b) {
   	1 => 3,
   	3 => 'ab',
   	5 => match($c) {
   		6 => new X,
   		7 => [],
   	}
   	default => false,
   };
   
   ?>

Suggestions
___________

* Merge the two match() in one.
* Replace the nested match call by a method call.
