String Interpolation Favorite
+++++++++++++++++++++++++++++

  This analysis collects the various ways that string interpolation is done inside strings. Until PHP 8.1, there were 4 ways :

.. code-block:: php
   
   <?php
   
   $a = "$variable";
   $a = "$object->property";
   $a = "$array[index]";
   
   $a = "";
   $a = "{$variable}";
   
   $a = "";
   ?>
