Empty Loop
++++++++++

  Empty loop may come from a typo, where a semi colon detach the block from its loop. 


.. code-block:: php
   
   <?php
   
   $i = 0;
   // sneaky ; 
   while($i < 10) ; {
   	$i++;
   }
   
   // another sneaky semicolon
   foreach($a as $b) ; 
   {
   	$i++;
   }
   
   ?>

Suggestions
___________

* Remove the extra semicolon
* Fill the loop with a payload
