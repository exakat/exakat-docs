Resources Usage
+++++++++++++++

  List of situations that are creating resources.

.. code-block:: php
   
   <?php
       // This functioncall creates a resource to use
       $fp = fopen('/tmp/file.txt', 'r');
       
       if (!is_resource($fp)){
           thrown new RuntimeException('Could not open file.txt');
       }
   ?>
