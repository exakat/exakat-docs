No Need For Triple Equal
++++++++++++++++++++++++

  There is no need for the identity comparison when the methods returns the proper type.

.. code-block:: php
   
   <?php
   
   // foo() returns a string. 
   if ('a' === foo()) {
       // doSomething()
   }
   
   
   function foo() : string { 
       return 'a';
   }
   
   ?>
