Use str_starts_with()
+++++++++++++++++++++

  There is a dedicated function to check the prefix of a string. 


.. code-block:: php
   
   <?php
   
   if (str_starts_with($a, 'abc')) { }
   
   // Before PHP 8.2
   if (substr($a, 0, 3) === 'abc') { }
   
   ?>

See also `str_ends_with() <https://www.php.net/str_ends_with>`_.


Suggestions
___________

* Use the native PHP function
