Multiple Catch
++++++++++++++

  Indicates if a try structure have several catch statement.


.. code-block:: php
   
   <?php
   
   // This try has several catch
   try {
       doSomething();
   } catch (RuntimeException $e) {
       processRuntimeException();
   } catch (OtherException $e) {
       processOtherException();
   }
   
   ?>
