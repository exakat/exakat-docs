Useless Switch
++++++++++++++

  This switch has only one case. It may very well be replaced by a ifthen structure.


.. code-block:: php
   
   <?php
   switch($a) {
       case 1:
           doSomething();
           break;
   }
   
   // Same as 
   
   if ($a == 1) {
       doSomething();
   }
   ?>

Suggestions
___________

* Turn the switch into a if/then for better readability
* Add other cases to the switch, making it adapted to the situation
