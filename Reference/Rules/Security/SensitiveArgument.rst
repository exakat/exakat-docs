Sensitive Argument
++++++++++++++++++

  Spot the argument that are sensitive for security. The functioncalls that are hosting a sensitive argument are called a sink.


.. code-block:: php
   
   <?php
   
   // first argument $query is a sensitive argument 
   mysqli_query($query);
   
   ?>
