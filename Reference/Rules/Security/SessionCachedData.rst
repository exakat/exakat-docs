Unvalidated Data Cached In Session
++++++++++++++++++++++++++++++++++

  Data is cached in the $_SESSION variable and later reused. 


.. code-block:: php
   
   <?php
   
   $_SESSION['a'] = $_GET['a'];
   
   
   // across the code, this call
   function foo() {
   	echo $_SESSION["a"];
   }
   
   ?>

Suggestions
___________

* Validate data before storing in the SESSION
