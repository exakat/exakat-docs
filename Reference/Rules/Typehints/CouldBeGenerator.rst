Could Be Generator
++++++++++++++++++

  Return value may be typed ``generator``.


.. code-block:: php
   
   <?php
   
   // Yield makes foo() a generator
   function foo() {
       yield 1; 
       // Returns an int
       return $b + 8;
   }
   
   ?>

See also `class <https://www.php.net/manual/en/language.oop5.basic.php#language.oop5.basic.class>`_.


Suggestions
___________

* Add `\Generator` typehint to the method.
