Type Could Be Never
+++++++++++++++++++

  Mark return types that can be set to ``never``.

.. code-block:: php
   
   <?php
   
   function foo($b) {
       // this function never returns
       die();
   }
   
   ?>

Suggestions
___________

* Add the 'never' returntype
