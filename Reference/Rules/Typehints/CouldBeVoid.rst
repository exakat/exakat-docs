Could Be Void
+++++++++++++

  Mark return types that can be set to void.


.. code-block:: php
   
   <?php
   
   // No return, this should be void.
   function foo() {
       ++$a; // Not useful
   }
   
   ?>


All abstract methods (in classes or in interfaces) are omitted here.

Suggestions
___________

* Add the void typehint to the code.
