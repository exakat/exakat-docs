Could Be String
+++++++++++++++

  Mark arguments, properties, constants and return types that can be set to ``string``.

.. code-block:: php
   
   <?php
   
   // Accept a string as input 
   function foo($a) {
       // Returns a string
       return $a . 'string';
   }
   
   ?>

Suggestions
___________

* Choose the string typehint, and add it to the code.
