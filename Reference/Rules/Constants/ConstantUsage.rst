Constants Usage
+++++++++++++++

  List of constants being used.


.. code-block:: php
   
   <?php
   
   const MY_CONST = 'Hello';
   
   // PHP_EOL (native PHP Constant)
   // MY_CONST (custom constant)
   echo PHP_EOL . MY_CONST;
   
   ?>
