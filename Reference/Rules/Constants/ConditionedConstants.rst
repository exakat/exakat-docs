Conditioned Constants
+++++++++++++++++++++

  Indicates if a constant will be defined only if a condition is met.


.. code-block:: php
   
   <?php
   
   if (time() > 1519629617) {
       define('MY_CONST', false);
   } else {
       define('MY_CONST', time() - 1519629617);
   }
   
   ?>
