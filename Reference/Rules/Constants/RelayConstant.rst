Relay Constant
++++++++++++++

  A relay constant is a constant that gives another name to an existing constant.

.. code-block:: php
   
   <?php
   
   class A {
   	const A = 1;
   }
   
   class B {
   	// this constant is actually the A::A constant 
   	const B = A::A;
   }
   
   ?>
