Constants Names
+++++++++++++++

  List of PHP defined global constants in the source code. Constants are defined with the ``define()`` functioncall or ``const`` command. 

.. code-block:: php
   
   <?php
   
   // with const
   const X = 1;
   
   // with define()
   define ('Y', 2);
   
   ?>

See also `PHP Constants <https://www.php.net/manual/en/language.constants.php>`_.
