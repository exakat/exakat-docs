Is PHP Constant
+++++++++++++++

  Mark a constant if it is a PHP constant.

.. code-block:: php
   
   <?php
   
   // This is an PHP constant
   $a = HTML_ENTITIES;
   
   // This is an PHP function
   $a = CMS_ORDER;
   
   ?>
