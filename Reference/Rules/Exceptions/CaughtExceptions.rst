Caught Exceptions
+++++++++++++++++

  This rule collects the exceptions used in catch clause. Those are the caught exceptions.

.. code-block:: php
   
   <?php
   
   try {
       foo();
   } catch (MyException $e) {
       fixException();
   } catch (MyOtherException1|MyOtherException2) {
       fixException();
   } finally {
       clean();
   }
   
   ?>
