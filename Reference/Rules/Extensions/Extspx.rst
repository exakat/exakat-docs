ext/spx
+++++++

  SPX, which stands for Simple Profiling eXtension, is just another profiling extension for PHP. 


.. code-block:: php
   
   <?php
   
   while ($task = get_next_ready_task()) {
     spx_profiler_start();
     try {
       $task->process();
     } finally {
       spx_profiler_stop();
     }
   }
   ?>

See also `<https://github.com/NoiseByNorthwest/php-spx>`_.
