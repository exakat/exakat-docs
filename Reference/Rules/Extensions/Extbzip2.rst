ext/bzip2
+++++++++

  Extension ext/bzip2.

Bzip2 Functions for PHP.

.. code-block:: php
   
   <?php
   
   $file = '/tmp/foo.bz2';
   $bz = bzopen($file, 'r') or die('Couldn\'t open $file for reading');
   
   bzclose($bz);
   
   ?>

See also `Bzip2 Functions <https://www.php.net/bzip2>`_ and `bzip2 <https://en.wikipedia.org/wiki/Bzip2>`_.
