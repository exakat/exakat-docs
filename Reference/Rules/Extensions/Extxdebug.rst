ext/xdebug
++++++++++

  Xdebug extension.

The Xdebug is a extension PHP which provides debugging and profiling capabilities.

.. code-block:: php
   
   <?php
   class Strings
   {
       static function fix_string($a)
       {
           echo
               xdebug_call_class().
               "::".
               xdebug_call_function().
               " is called at ".
               xdebug_call_file().
               ":".
               xdebug_call_line();
       }
   }
   
   $ret = Strings::fix_string( 'Derick' );
   ?>

See also `Xdebug <https://xdebug.org/>`_.
