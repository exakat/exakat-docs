Globals
+++++++

  Global variables.

.. code-block:: php
   
   <?php
   
   // global via global keyword
   global $a, $b;
   
   // global via $GLOBALS variable
   $GLOBALS['c'] = 1;
   
   ?>

See also `Global keyword <https://www.php.net/manual/en/language.variables.scope.php#language.variables.scope.global>`_.
