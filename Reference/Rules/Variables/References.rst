Variable References
+++++++++++++++++++

  Variables that are holding references.

References are created with ``=&`` operators, and later propagated with the same operators, or via reference-arguments.


.. code-block:: php
   
   <?php
   
   $a = '1'; // not a reference
   $b = &$a; // a reference
   
   ?>

See also `References <https://www.php.net/references>`_.
