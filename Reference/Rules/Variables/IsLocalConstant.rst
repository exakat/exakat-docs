Variable Is A Local Constant
++++++++++++++++++++++++++++

  A variable that is written once, then never modified : it behaves like a constant. Some other rule may take advantage of this.


.. code-block:: php
   
   <?php
   
   function foo() {
       $localConstant = 'Hello';
       echo $localConstant;
   
       $variable = 'Hello, ';
       $variable .= date('r');
       echo $variable;
   }
   
   ?>
