Make Class Constant Definition
++++++++++++++++++++++++++++++

  This command adds DEFINITION link between class constant definitions and their usage.

.. code-block:: php
   
   <?php
   
   class x {
       public const A = 1;
   }
   
   // Link to the constant definition
   echo x::A;
   
   // Cannot find the original class
   echo $x::A;
   
   ?>
