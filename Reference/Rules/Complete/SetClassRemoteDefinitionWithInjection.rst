Set Class Remote Definition With Injection
++++++++++++++++++++++++++++++++++++++++++

  Links a method call and its definition, thanks to a typehint.


.. code-block:: php
   
   <?php
   
   class A {
       function goo() {}
   }
   
   function foo(A $arg) {
       // This goes to method A::goo(), thanks to the typehint
       $arg->goo();
   }
   
   ?>
