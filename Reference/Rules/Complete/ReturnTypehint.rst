Add Return Typehint
+++++++++++++++++++

  Add obvious returntypes to methods, functions, closures and arrow functions.


.. code-block:: php
   
   <?php
   
   // This has no type, but could use int
   function foo() {
       return 1;
   }
   
   ?>
