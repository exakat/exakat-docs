Set Class Property Definition With Typehint
+++++++++++++++++++++++++++++++++++++++++++

  Links method call to its definition, thanks to property typehinting. The link is ``DEFINITION``.


.. code-block:: php
   
   <?php
   
   class x {
       public x $p = null;
   
       public function bar() {
           return $this;
       }
   }
   
   $x = new x;
   
   // $x->p is of 'x' class
   $x->p->bar();
   
   ?>
