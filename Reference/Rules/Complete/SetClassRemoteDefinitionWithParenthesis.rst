Set Class Remote Definition With Parenthesis
++++++++++++++++++++++++++++++++++++++++++++

  Links method call to its definition, thanks to the new in parenthesis. The link is ``DEFINITION``.


.. code-block:: php
   
   <?php
   
   class x {
       public function bar() {    }
   }
   
   function foo() {
       // This links to class x, method bar(), thanks to the new.
       return (new x)->bar();
   }
   
   ?>
