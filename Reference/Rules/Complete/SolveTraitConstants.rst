Solve Trait Constants
+++++++++++++++++++++

  Adds a link between `static <https://www.php.net/manual/en/language.oop5.static.php>`_ constant usage and a class constant set in a trait. 


.. code-block:: php
   
   <?php
   
   trait t { const A = 1; }
   
   class x {
   	use t;
   	
   	function foo() {
   		echo self::A;
   	}
   }
   
   ?>


Constants in traits are added in PHP 8.2.
