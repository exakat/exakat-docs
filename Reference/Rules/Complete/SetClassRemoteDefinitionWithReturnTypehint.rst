Set Class Remote Definition With Return Typehint
++++++++++++++++++++++++++++++++++++++++++++++++

  Links method call to its definition, thanks to the typed return. The link is ``DEFINITION``.


.. code-block:: php
   
   <?php
   
   class x {
       public function bar() {    }
   }
   
   function foo() {
       $a = bar();
       // This links to class x, method bar(), thanks to the new.
       return $a->bar();
   }
   
   function bar() : x {
       return new x;
   }
   
   ?>
