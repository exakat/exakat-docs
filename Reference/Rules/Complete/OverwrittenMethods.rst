Overwritten Methods
+++++++++++++++++++

  This command adds OVERWRITE link between methods definitions of classes.


.. code-block:: php
   
   <?php
   
   class x {
       protected function foo() {}
   }
   
   class y extends x {
       protected function foo() {}
   }
   
   ?>


The `foo` method will be linked between classes x and y, with an OVERWRITE link.
