Create Foreach Default
++++++++++++++++++++++

  This command adds DEFAULT link from the blind variables to the literal available.

.. code-block:: php
   
   <?php
   
   // $a may b e 1, 2 or 3
   foreach([1,2,3] as $a) {
       echo $a;
   }
   
   ?>
