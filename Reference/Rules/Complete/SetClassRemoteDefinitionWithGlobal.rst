Set Class Remote Definition With Global
+++++++++++++++++++++++++++++++++++++++

  Links method call to its definition, thanks to the global definition. The link is ``DEFINITION``.


.. code-block:: php
   
   <?php
   
   class x {
       public function bar() {    }
   }
   
   global $a;
   $a = new X;
   
   function foo() {
       global $a;
       
       // This links to class x, method bar(), thanks to global.
       return $a->bar();
   }
   
   ?>
