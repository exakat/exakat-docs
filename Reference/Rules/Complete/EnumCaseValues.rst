Enum Case Values
++++++++++++++++

  Adds a `DEFINITION`link between a `static <https://www.php.net/manual/en/language.oop5.static.php>`_ Enumeration case and its actual defined value. 

No link is added when no value is defined.

.. code-block:: php
   
   <?php
   
   enum E: string {
      case Foo = 'foo';
   }
   
   // Constants
   const C = E::Foo->name;  
   
   ?>
