Set Method Fnp
++++++++++++++

  Complete code by adding the ``fullnspath`` property to methods calls. 

It makes it faster to find definitions later.

.. code-block:: php
   
   <?php
   
   function foo(X $a) {
   	// \x::moo 
   	$a->moo();
   }
   
   ?>
