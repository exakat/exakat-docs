Overwritten Properties
++++++++++++++++++++++

  This command adds OVERWRITE link between property definitions of classes.


.. code-block:: php
   
   <?php
   
   class x {
       protected $p = 1;
   }
   
   class y extends x {
       protected $p = 1;
   }
   
   ?>


The `$p` property will be linked between classes x and y, with an OVERWRITE link.
