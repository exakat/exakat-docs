Php Native Reference Variable
+++++++++++++++++++++++++++++

  Native functions, such as `sort() <https://www.php.net/sort>`_ (first argument), or `preg_match_all() <https://www.php.net/preg_match_all>`_ (third argument), use reference.

.. code-block:: php
   
   <?php
   
   $a = [3,1,2];
   sort($a);
   $a === [1,2,3];
   
   ?>
