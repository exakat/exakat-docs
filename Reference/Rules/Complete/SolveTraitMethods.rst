Solve Trait Methods
+++++++++++++++++++

  This command adds DEFINITION link between trait's method definitions and their usage in classes.

.. code-block:: php
   
   <?php
   
   trait t {
       function foo() {
       
       }
   }
   
   class x {
       use t { t::foo as foo2; };
       
       function bar() {
           // Link to foo() in trait t
           $this->foo();
           // Link to foo() in trait t, thanks to 'as'
           $this->foo2();
       }
   }
   
   ?>
