Array Index
+++++++++++

  List of all indexes used in arrays. 

.. code-block:: php
   
   <?php
   
   // Index
   $x['index'] = 1;
   
   // in array creation
   $a = array('index2' => 1);
   $a2 = ['index3' => 2];
   
   ?>
