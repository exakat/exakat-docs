Mass Creation Of Arrays
+++++++++++++++++++++++

  Literal creation of an array, by assigning a lot of index.

.. code-block:: php
   
   <?php
       
   $row['name'] = $name;
   $row['last'] = $last;
   $row['address'] = $address;
   
   ?>
