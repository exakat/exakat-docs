Wrong Case Namespaces
+++++++++++++++++++++

  Namespaces are case-insensitive.

.. code-block:: php
   
   <?php
   
   // Namespaces should share the same case
   namespace X {}
   
   namespace x {}
   
   ?>

Suggestions
___________

* Synchronize all names
