Undefined Enumcase
++++++++++++++++++

  The enumeration case does not exists. It may also be a constant.

.. code-block:: php
   
   <?php
   
   enum theEnum {
       case A; // an enum case
       
       // a constant
       const C = 1;
   }
   
   function foo(theEnum $a) {}
   
   foo(theEnum::A);
   foo(theEnum::C);
   
   ?>
