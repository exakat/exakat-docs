Global Code Only
++++++++++++++++

  This rule reports files that only contain global code.

.. code-block:: php
   
   <?php
   
   include 'another_file.pnp';
   
   // This sets an options, but does not execute anything
   set_memory_limit(-1);
   
   // Some definitions, no code 
   const A = 1;
   
   function foo() {}
   
   class x {}
   
   ?>
