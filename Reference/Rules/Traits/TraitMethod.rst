Trait Methods
+++++++++++++

  List the names of the methods in a trait. 

.. code-block:: php
   
   <?php
   
   trait t {
       private $property = 1;
       
       // This is a trait method name
       function foo() {
           // This is not a trait method 
           return function($a) { return $a + 1; }
       }
   }
   
   ?>
