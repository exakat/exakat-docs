Empty Traits
++++++++++++

  List of all empty trait defined in the code. 



Such traits may be reserved for future use. They may also be forgotten, and dead code.

.. code-block:: php
   
   <?php
   
   // empty trait
   trait t { }
   
   // Another empty trait
   trait t2 {
       use t; 
   }
   
   ?>

Suggestions
___________

* Add some code to the trait
* Remove the trait
