Traits Usage
++++++++++++

  Usage of traits in the code.


.. code-block:: php
   
   <?php
   
   trait t {
       function t() {
           echo 'I\'m in t';
       }
   }
   
   class foo {
       use t;
   }
   
   $x = new foo();
   $x->t();
   
   ?>

See also `Traits <https://www.php.net/manual/en/language.oop5.traits.php>`_.
