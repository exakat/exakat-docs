Enum Usage
++++++++++

  This rule reports usage of PHP native enumerations. Enumerations were introduced in PHP 8.1 and are not backward compatible.

.. code-block:: php
   
   <?php
   
   enum X {
       case A;
       case B;
   }
   
   foo(X::A);
   
   ?>

See also `Enumerations in PHP <https://www.delftstack.com/howto/php/php-enum/>`_.
