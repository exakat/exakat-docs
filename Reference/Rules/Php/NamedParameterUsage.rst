Named parameters works for both custom methods and PHP native functions.

.. code-block:: php
   
   <?php
   
   // named parameters
   foo(a : 1, b : 2);
   foo(b : 2, a : 1);
   
   // positional parameters
   foo(1, 2);
   
   function foo($a, $b) { }
   
   ?>
