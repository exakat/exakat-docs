Dl() Usage
++++++++++

  Dynamically load PHP extensions with `dl() <https://www.php.net/dl>`_.

.. code-block:: php
   
   <?php
   
       // dynamically loading ext/vips
   	dl('vips.' . PHP_SHLIB_SUFFIX);
   
   ?>

See also `dl <http://www.php.net/dl>`_.
