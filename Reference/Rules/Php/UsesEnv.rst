Uses Environment
++++++++++++++++

  This rule spots usage of ``$_ENV``, ``getenv()`` and ``putenv()`` functions: they fetch data from the environment variables.

.. code-block:: php
   
   <?php
   
   // Take some configuration from the environment
   $secret_key = getenv('secret_key');
   
   ?>
