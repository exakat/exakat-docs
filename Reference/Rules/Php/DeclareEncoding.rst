Encoding Usage
++++++++++++++

  Usage of declare(encoding = );.

.. code-block:: php
   
   <?php
   
   // Setting encoding for the file;
       declare(encoding = 'UTF-8');
   
   ?>

See also `declare <https://www.php.net/manual/en/control-structures.declare.php>`_.
