Use Enum Case In Constant Expression
++++++++++++++++++++++++++++++++++++

  Enum cases are constants, and may be used in constant definitions, as value. This is valid both with the case itself, or with their value, for the backed enum version.

.. code-block:: php
   
   <?php
   
   enum A {
   	case A;
   }
   
   enum B : string {
   	case B = 'b';
   }
   
   class C {
   	const C1 = A::A;
   	const C2 = B::B->value;
   }
   ?>
