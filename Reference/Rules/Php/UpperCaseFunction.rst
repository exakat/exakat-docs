Unusual Case For PHP Functions
++++++++++++++++++++++++++++++

  Usually, PHP functions are written all in lower case.

.. code-block:: php
   
   <?php
   
   // All uppercases PHP functions
   ECHO STRTOLOWER('This String');
   
   ?>
