Try With Multiple Catch
+++++++++++++++++++++++

  Try may be used with multiple catch clauses.

.. code-block:: php
   
   <?php
   
   try { 
       OneCatch(); 
   } catch (FirstException $e) {
   
   }
   
   try { 
       TwoCatches(); 
   } catch (FirstException $e) {
   } catch (SecondException $e) {
   }
   
   ?>

See also `Exceptions <https://www.php.net/manual/en/language.exceptions.php>`_.
