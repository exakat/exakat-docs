A new type was introduced : ``never``. 

It can't be used before PHP 8.1, and will be confused with classes or interfaces.

.. code-block:: php
   
   <?php
   
   function test() : never
   {
       exit();
   }
   
   ?>

See also  `PHP RFC: noreturn type <https://wiki.php.net/rfc/noreturn_type>`_.
