Pear Usage
++++++++++

  Pear Usage : list of Pear packages in use.

.. code-block:: php
   
   <?php
       require_once('MDB2.php');
       $dsn = 'mysql://user:pass@host';
       $mdb2 = &MDB2::factory($dsn);
       $mdb2->setFetchMode(MDB2_FETCHMODE_ASSOC);
   ?>
