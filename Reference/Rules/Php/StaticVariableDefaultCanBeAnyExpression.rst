Indeed, those are variables, so their value can be build from other variables, and even functioncalls. 

This feature was introduced in PHP 8.3.

.. code-block:: php
   
   <?php
   
   function foo($init) {
   	static $variable = foo($a);
   	
   	return $variable++;
   }
   ?>
