Mixed Typehint Usage
++++++++++++++++++++

  Usage of the mixed typehint.

.. code-block:: php
   
   <?php
   
   function foo() : mixed {
       switch(rand(0, 3)) {
           case 0:
               return false;
               
           case 1: 
               return 'a';
               
           case 2:
               return [];
               
           default:
               return null;
       }
   }
   
   ?>

See also `Type declarations <https://www.php.net/manual/en/language.types.declarations.php>`_.
