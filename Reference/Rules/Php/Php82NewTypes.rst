PHP 8.2 New Types
+++++++++++++++++

  This rule reports usage of the new PHP 8.2 types. This is the `true` type.

This type is not available before version 8.2.

.. code-block:: php
   
   <?php
   
   function foo() : true { 
   	return true;
   }
   
   ?>
