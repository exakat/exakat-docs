PHP 7.4 Removed Directives
++++++++++++++++++++++++++

  List of directives that are removed in PHP 7.4.

+ allow_url_include

See also `Deprecation allow_url_include <https://wiki.php.net/rfc/deprecations_php_7_4#allow_url_include>`_.


Suggestions
___________

* Stop using this directive
