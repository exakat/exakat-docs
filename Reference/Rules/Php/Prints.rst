Displays Text
+++++++++++++

  Function calls that displays something to the output.

.. code-block:: php
   
   <?php
   
   // Displays de the content of $a
   print $a;
   
   // Displays de the content of $b
   print_r($b);
   
   // Returns de the content of $b, no display.
   $c = var_export($b, true);
   
   ?>
