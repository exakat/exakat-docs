Directives Usage
++++++++++++++++

  List of the directives mentioned in the code.


.. code-block:: php
   
   <?php
   
   //accessing the configuration to change it
   ini_set('timelimit', -1);
   
   //accessing the configuration to check it
   ini_get('safe_mode');
   
   ?>

See also `ini_set() <https://www.php.net/ini_set>`_.
