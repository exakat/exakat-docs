Setting keys when using `list() <https://www.php.net/list>`_ is a PHP 7.1 feature.

.. code-block:: php
   
   <?php
   
   // PHP 7.1 and later only
   list('a' => $a, 'b' => $b) = ['b' => 1, 'c' => 2, 'a' => 3];
   
   ?>
