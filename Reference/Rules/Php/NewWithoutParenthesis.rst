In PHP 8.4, it is possible to reach a property or call a method right on the ``new`` call, without putting the new inside parenthesis.

This syntax is not backward compatible.

.. code-block:: php
   
   <?php
   
   echo new X()->property;
   
   echo new X()->method();
   
   ?>
