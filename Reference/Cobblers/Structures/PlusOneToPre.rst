Plus One To Pre Plusplus
++++++++++++++++++++++++
Transforms a `+ 1` or `- 1` operation into a plus-plus (or minus-minus).

.. _plus-one-to-pre-plusplus-before:

Before
______
.. code-block:: php

   <?php
       $a = $a + 1;
   ?>

.. _plus-one-to-pre-plusplus-after:

After
_____
.. code-block:: php

   <?php
       ++$a;
   ?>
