Array To Bracket
++++++++++++++++
This cobbler updates the array() syntax, and changes it to the bracket syntax.


.. _array-to-bracket-before:

Before
______
.. code-block:: php

   <?php
   $a = array(1, 2, 3);
   ?>

.. _array-to-bracket-after:

After
_____
.. code-block:: php

   <?php
   $a = [1, 2, 3];
   ?>
